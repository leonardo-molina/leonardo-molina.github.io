Autonomous Maze-Navigating Robot with Robotic Manipulation

An autonomous robot capable of navigating through a maze, identifying a target block, and using a robotic arm to pick it up and place it onto a designated shelf. Designed and built from the ground up, this project integrates embedded control, sensor fusion, path-planning algorithms, and robotic manipulation.

My Contribution

  • Served as the Software Lead, developing the robot’s full control software stack in Python on a Raspberry Pi.
  • Implemented autonomous navigation, motion control, and system logic.
  • Integrated Adafruit I²C sensors—including color sensors—for block detection and navigation cues.
  • Contributed to hardware development, helping design and build the chassis and robotic arm.

Tools Used

  • Raspberry Pi (primary controller)
  • Python (navigation, control, and sensor processing)
  • Adafruit I²C sensors + Adafruit color sensors
  • Motor drivers and servo controllers
  • I²C protocol for multi-sensor communication
